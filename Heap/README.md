## ヒープ

- 子ノードの値より親ノードの値のほうが大きいか等しい
  - または、小さいか等しい
- 配列の添字がiのとき、その配列要素の親ノードは(i - 1) / 2
- 配列の添字がiのとき、その配列要素の左の子ノードは2i + 1
- 配列の添字がiのとき、その配列要素の右の子ノードは2i + 2
