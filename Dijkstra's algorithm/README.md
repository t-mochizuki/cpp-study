## Dijkstra's algorithm

- 重み付き有向グラフの最短経路長を求めるアルゴリズム
  - 辺の重みは非負の整数
  - 任意の頂点から始め、各頂点への最短経路長を更新することを繰り返し、すべての頂点を辿る
- すべての辺の重みが同じときは幅優先探索での時間計算量はO(m)
  - 辺の数をmとする
