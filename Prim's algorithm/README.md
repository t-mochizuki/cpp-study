## Prim's algorithm

- 重み付き無向グラフの最小全域木を求めるアルゴリズム
- 任意の頂点から始め、重みが最小の辺を選ぶことを繰り返し、すべての頂点を辿る
  - 木を育てる: 木に含まれない辺から重みが最小の辺を選び、その辺を木に移す
